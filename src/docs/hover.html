Hover mask

A set of utilities that on hover scales up the background and shows the underlying content.

.hover-mask the main wrapper.

.hover-mask-image to set the background picture.

.hover-mask-content the empty container where you can add any content with any style.

.hover-mask-content-up effect for content to appear from top to bottom.

.hover-mask-content-down effect for content to appear from bottom to top.
