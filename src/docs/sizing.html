Responsive sizing

See Bootstrap docs

The following values: 10%, 20%, 25%, 30%, 40%, 50%, 60%, 70%, 75%, 80%, 90%, 100%, auto.

  • .w-{breakpoint}-{size}: sets the width in %.
  • .h-{breakpoint}-{size}: sets the height in %.
  • .vw-{breakpoint}-{size}: sets the width in vw.
  • .vh-{breakpoint}-{size}: sets the height in vh.
  • .min-vw-{breakpoint}-{size}: sets the min-width in vw.
  • .min-vh-{breakpoint}-{size}: sets the min-height in vh.
